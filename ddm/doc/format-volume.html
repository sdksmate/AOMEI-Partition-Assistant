Format Volume

The volume will be formatted automatically when it is created. When there are a lot of data and you don't want to delete this volume, format volume is a better choice. The volume will be change to a fresh volume after formatting. Format volume could also help you change the file system, the cluster size (usually it is default as 4KB, and we do not recommend changing it) and the label.

Format Volume Step by Step
Step1. Launch Aomei Dynamic Disk Manager, right-click the volume you want to format. Select "Format Volume".

Step2. In the pop-up window, you could set up the "file system", the "cluster size" and the "label". After setting up all the information, click "Finish".

Step3. Finally, click "Commit" to perform the operation.
