Change Label

The volume label is a short name that you can assign to a volume to better differentiate it from other volumes. Through set different labels to different volume, you could know the purpose of each volume easily, such as if the volume contains the system data, we could label it "System" and etc. Aomei Dynamic Disk Manger could help you achieve this easily.

Step1. Right-click the volume you want to set label. Select "Change Label".

Step2. Entering the name you want to set and click "OK"

Step2. Click "Commit" on the toolbar.

Note: You could change label during creating or formatting a volume.
